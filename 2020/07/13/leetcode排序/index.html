1. 获得第k个元素的几种思路

  • 排序

  • 快速选择（利用快速排序的partition()来实现）

  • 堆

    利用大顶堆来实现一个最小堆，我们的大顶堆里面存放的是最小的k个元素，其中堆顶元素即为第k大的值

2. 获得k个元素的思路

  • 堆排序

    假设我们要获得k个最小的元素，那就有两种思路：

    • 最小堆

      我们利用大顶堆来实现一个最小堆，即我们的堆里面存放的就是最小的k个元素，一直添加到堆中，当数目超过的时候，遍历元素小于堆顶元素，则弹出堆顶元素，并将此元素放入

    • 小顶堆

      我们一直遍历下去，留下来的最终结果就是最小的k个元素

  • 快速选择

    快速选择也可以求解 TopK Elements 问题，因为找到 Kth Element 之后，再遍历一次数组，所有小于等于 Kth Element 的元素都是 TopK Elements。

3. 数组中的第K个最大元素

  • 大顶堆

    维护一个size-k+1的大顶堆，当遍历到的元素小于堆顶元素的时候，则踢出堆顶元素，并插入新的元素

  • 小顶堆

    维护一个大小为k的小顶堆，当遍历到的元素大于堆顶元素的时候，则放进来
