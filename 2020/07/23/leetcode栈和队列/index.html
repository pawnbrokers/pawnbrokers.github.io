leetcode栈和队列

Created2020-07-23Updated2020-07-23

1. 用栈实现队列

  • 思路：

    • 队列要求的是先进先出，我们可以用两个栈来实现队列
    • 要注意的是在push的时候就要全部把他们放到数据栈中，但是为了保证新添加进来的元素也放到数据栈的最下方，所以需要先把数据栈中的元素拿出到stack1中，然后再在stack1中假如待添加的元素，再依次全部放入到数据栈中。

2. 用队列实现栈

  • 思路：
    • 类似上题，我们同样需要两个队列，也同样需要在push的时候将所有元素都放入到数据队列中
    • 只是由于栈要求的是后进来的元素在上面，所以我们需要先将待添加元素放入到临时队列中，再执行操作。

3. 最小栈

  • 思路：

    • 我们可否利用两个栈，一个存储的是正确的栈值，另一个存放当前状态下的最小值信息
    • 重点在于添加元素的时候，我们的最小值栈应该执行的操作。

4. 有效的括号

  • 思路：

    • 用栈来实现一一消除
    • 注意判空以及每次不匹配的话直接返回false

5. 每日温度

  • 思路1：

    • 方法1：暴力循环遍历（两次循环）

  • 思路2（单调栈）：

    • 维护一个单调递减的栈

    我们要找的是这一天之后的第一个比当前数字更大的数

    我们在进行一次遍历的时候，利用一个栈把元素也存储起来，加入遍历到当前元素，发现当前元素大于栈顶元素，则这个就是第一个大于他的数字
