1. Socket概述

Socket中文意思为插座的意思，专业术语称之为套接字，它把TCP/IP封装成了调用接口供开发者调用，也就是说开发者可以通过调用Socket相关API来实现网络通讯。在Java中也存在Socket相关API，主要分为两个，分别是基于UDP传输协议的Socket和基于TCP传输协议的Socket，本篇文章会对基于这两种传输协议的Socket进行详细描述。

2. UDP Socket

2.1 步骤

  • 发送方

    1. 创建UDP的socket服务
    2. 将要发送的数据放在DatagramSocket中，DatagramSocket会根据UDP协议对数据包、ip、端口号进行封装
    3. 通过udp的socket服务进行数据包发送
    4. 将udp服务关闭

  • 接收方

    1. 创建udp的socket服务，并明确自己的端口号
    2. 创建用来接收数据的数据包
    3. 将数据接收到数据包DatagramSocket中
    4. 通过DatagramSocket解析数据
    5. 管服务

4. TCP Socket

4.1 步骤

  • 发送方

    1. 创建TCP客户端Socket服务
    2. 与服务端进行连接
    3. 连接成功后，获取客户端socket输出流
    4. 通过输出流往服务端写入数据
    5. 关闭流

  • 服务端

    1. 创建服务端Socket并明确端口号
    2. 通过accept()方法获得客户端的Socket
    3. 通过客户端的socket获得输入流
    4. 通过输入流获得客户端传递的数据
    5. 关闭流
