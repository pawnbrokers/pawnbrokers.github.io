leetcode位运算

发表于2020-08-03更新于2020-08-03

0. 原理

基本原理

0s 表示一串 0，1s 表示一串 1。

0.1 按位异或

不同为1，相同为0，无非三种情况：

  • 与0按位异或，即不同的是1，相同的是0，所以与0异或之后依旧为原值
  • 与1按位异或，原来的1全部变成了0，原来的0全部变成了1，所以相当于按位取反
  • 与自己异或，自然每一位都一致，所以结果为0

利用 x ^ 1s = ~x 的特点，可以将一个数的位级表示翻转；利用 x ^ x = 0 的特点，可以将三个数中重复的两个数去除，只留下另一个数。

0.2 按位与

同样是三种情况：

  • 与0按位与，只有全是1的才为1，所以结果是0
  • 与1按位与，还是原值
  • 与自己与，依旧是原值

利用 x & 0s = 0 和 x & 1s = x 的特点，可以实现掩码操作。一个数 num 与 mask：00111100 进行位与操作，只保留 num 中与 mask 的 1 部分相对应的位。

0.3 位与技巧

n&(n-1) 去除 n 的位级表示中最低的那一位 1。例如对于二进制表示 01011011，减去 1 得到 01011010，这两个数相与得到 01011010。即去除n的最后一个1

n&(-n) 得到 n 的位级表示中最低的那一位 1。-n 得到 n 的反码加 1，也就是 -n=~n+1。例如对于二进制表示 10110100，-n 得到 01001100，相与得到 00000100。

0.4 mask技巧

要获取 111111111，将 0 取反即可，~0。

要得到只有第 i 位为 1 的 mask，将 1 向左移动 i-1 位即可，1<<(i-1) 。例如 1<<4 得到只有第 5 位为 1 的 mask ：00010000。

要得到 1 到 i 位为 1 的 mask，(1<<i)-1 即可，例如将 (1<<4)-1 = 00010000-1 = 00001111。

要得到 1 到 i 位为 0 的 mask，只需将 1 到 i 位为 1 的 mask 取反，即 ~((1<<i)-1)。

1. 汉明距离

  • 思路：

    • 不同的自然就是异或，这样我们就得到了所有的1，统计个数直接采用Integer.bitCount()

  • 如果不采用api，可以使用异或的值与1与，然后依次右移一位

2. 只出现一次的数字

  • 思路：

    • 我们自然可以用哈希表
    • 这里也可以想到x^x = 0, 0^x = x

3. 缺失数字

  • 思路1：

    • 我们同样可以利用哈希，但是这样的话就不是额外常数空间

  • 思路2：

    • 两次遍历自然也是可行的，又或者先排序再二分

  • 思路3：

    • 位运算的话需要从哪里入手呢？
    • 我们想象一下，排序之后相当于下标为0n-1，对应的元素为0n中且少一个，所以一共n个元素，我们再给他加一个0构成完整的n+1个数字，完成 两次异或自己为0，同时0异或其他的都为原值

4. 只出现一次的数字III

  • 思路1：

    • 哈希表

  • 思路2：

    • 异或自己的话就变为0，同时0异或他人都是自身，但是这里我们有两个元素，应该如何处理呢？

    • 我们异或一轮的话剩下的是3^5

    • 将数组的所有元素异或得到的结果为不存在重复的两个元素异或的结果。
